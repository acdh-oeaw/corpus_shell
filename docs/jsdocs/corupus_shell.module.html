CreateNewSearchPanel(configIdx, searchStr)

purpose: creates a new search panel and displays it.

  1. Creates a new div and styles it as needed by the ui. Its id is set to the unique panel id.
  2. Creates the "window title bar" using module:corpus_shell~Panel#GeneratePanelTitle
  3. Builds a query to be executed if some search string is given.
  4. Creates the ui elements for searching.
    -> module:corpus_shell~Panel#GenerateSearchInputs
    -> module:corpus_shell~Panel#GenerateSearchNavigation
  5. Adds a div for search results. -> module:corpus_shell~Panel#GenerateSearchResultsDiv
  6. Makes the panel draggable. -> module:corpus_shell~Panel#InitDraggable
Parameters:
Name Type Description
configIdx number Preselected index of SearchCombo
searchStr string Search string to inserted in search input

FillInPanelTitle(context, titlestring, pin, pinned)

Parameters:
Name Type Description
context jQuery The jQuery object that should be manipulated. This object is expected to have some space of class c_s-ui-widget-header-title for the title and two hidden markers of class titletopiconpin for the pinned state one of which has also class c_s-greyed and both have class c_s-hidden initially. The class c_s-hidden will be removed to denote the current state. Elements with class titletopiconclose and titletopiconmax will have onclick handlers attached.
titlestring string The panel's title.
pin number With a value of 1 the panel gets a pin icon.
pinned boolean Boolean value that tells wether the panel is pinned. purpose: creates a paragraph containing the panel title and the icons to pin (optional), maximize, and close.
